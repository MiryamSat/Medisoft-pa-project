# Medisoft PA

Simulazione di un gestionale sanitario con focus sul customer care e la gestione delle richieste dei pazienti.

## Funzionalità

- Gestione richieste pazienti
- Stati: Aperto, In lavorazione, Chiuso
- Filtro richieste
- Integrazione con database MySQL

## Tecnologie

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Obiettivo

Simulare il lavoro di un team di customer care sanitario, con gestione delle richieste e organizzazione operativa.
